# hyperdrive-ui

Explore the contents of a [hyperdrive](http://github.com/mafintosh/hyperdrive) in the browser.

Live demo: [http://karissa.github.io/hyperdrive-ui/](http://karissa.github.io/hyperdrive-ui/)
